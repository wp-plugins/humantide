== Description ==

The social platform of opinion in real time.

What is a tide? ...

... Is a query, an event, a feeling, a cause. 
Everything what you want to look for support or opinion of the people you care about. 
Friends, colleagues, partners, employees, collaborators ... in short 'yours'.


What is a survey? ...

... Is a set of tides.
If you want to know what is the favorite, create your Poll Top Rated
If you want to ask several questions at once, create your Multi Survey


== Installation ==

1. Upload humantide folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Login or create an user in [homepage](https://humantide.com)

For basic usage, you can also have a look at the plugin [homepage](https://humantide.com).

== Frequently Asked Questions ==

= What is a tide? =

Is a query, an event, a feeling, a cause. Everything what you want to look for support or opinion of the people you care about.

= What is a survey? =

Is a set of tides.

= How it works, and what can I do with this plugin? =

You need to be registered in [homepage](https://humantide.com) to be able to use the plugin. Then, you can create tides or surveys with the plugin, and they will be post in your blog.
You can see all your tides and/or surveys through the plugin or in [homepage](https://humantide.com).
You can also make use of our widget to show any tide or survey in your blog.
Make sure to publish your content public if you are going to post it in your blog ;)

= Is a widget available? =

Yes. There is a widget for tides and surveys.

= How does the widget works? =

It needs the url of the tide or the survey generated through [homepage](https://humantide.com). You can customize different options to adapt it to your page
